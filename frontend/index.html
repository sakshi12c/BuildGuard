⚙️ How It Works
📤

1. Upload

Upload your C source code file. It is temporarily stored for analysis only.

⚙️

2. Double Compile

BuildGuard compiles your code twice using GCC — once with a locked timestamp, once without.

🔍

3. Compare

We compare both outputs byte-by-byte using SHA-256 hashing and give you a trust score.

📊

4. Results

See your reproducibility score, hash comparison, and exactly what caused any differences.

💡

5. Fix Suggestions

Get actionable advice on how to make your build fully reproducible and secure.

📜

6. History

All your past checks are saved so you can track improvements over time.
